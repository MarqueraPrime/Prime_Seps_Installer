# PrimeSeps 2026 — Instalador

Instalador gráfico para o **PrimeSeps 2026**, uma variação do QuikSeps para Adobe Photoshop 2025+.

---

## Como usar

1. Baixe o arquivo **`PrimeSeps_Installer_v1.0.0.exe`**
2. Execute com **Administrador** (botão direito → "Executar como administrador")
3. O instalador detecta automaticamente a pasta do Photoshop
4. Clique em **Instalar Agora**
5. Pronto! Os arquivos serão copiados para:
   - `Presets/Actions/`
   - `Presets/Color Books/`
   - `Presets/Color Swatches/`

> O instalador é auto-contido (\~12 MB). Não precisa de Python instalado.

## Ativação no Photoshop

1. **Window > Actions** → Load Actions → `PrimeSeps.atn`
2. **Edit > Presets > Preset Manager** → Load → `PrimeSeps Custom Color.csf`
3. Abra sua imagem e clique duas vezes no **ícone de corrente** (chain) do canal **Magenta** para destravar a Layer 1

## Requisitos

- Windows 10 ou 11
- Adobe Photoshop 2025 ou superior
- Permissão de administrador para instalar
